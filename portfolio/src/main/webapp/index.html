Following are my technical projects:

1) Portfolio website with blog (Year1, Semester 2 project) Made a portfolio website with login system and blogging facility as a part of “Fundamentals of web technology” module in Uni.

Platforms/Languages: HTML, CSS, Javascript, PHP, MySQL database

Key Characteristics:

  • Has HTML semantic structural elements.
  • Blog and login system connect to a database to edit and get information.
  • Used JavaScript for event processing of buttons.

2) Gaming application on Google Cloud (Year1, Semester 2 project) Developed a Car Racing Stimulation Game and deployed it on Google Cloud platform with high availability, autoscaling and Global load balancing features.

Platforms/Languages: Java, Linux, JOption pane, ArrayLists, Exception handling, Object-Oriented Programming, BlueJ, JFrame, Java awt and swing classes.

Key Characteristics:

  • Graphical user interface using JOption pane and JFrame.
  • High Availability across multiple GCP regions.
  • Autoscaling based on spikes in gaming activity.
  • Global load balancing to distribute traffic across regions.
  • Redundancy/Failover features to avoid downtime.
  • Uses polymorphism, late dynamic binding and substitution principle.
  • By using file Input/ Output, game progress can be saved.

3) Traffic Lights Controller This project is to implement traffic light controller by integrating electronics with computer programming. It can be extended to interface with other devices forming the basis of IoT.

This project constitutes of two parts Traffic lights circuit integration with Raspberry PI Design and implementation of software for traffic light controller algorithm

Hardware Platform: Raspberry Pi3 Model B. Operating System: Raspbian Language: Scratch Programming Interface

Key Characteristics:

  • Traffic light LED circuit.
  • Programmable traffic light controller interface.
  • Extensible architecture.

4) Set Theorem Prover Theorem prover is a tool to verify various set theorems. It can be used by students/examiners/mathematicians to confirm if the set equations are valid or not. It’s also a fun way of learning core functionalities of sets. It’s an extension of “Set Calculator”(a mini project before this) which has a “Theorem Proving Engine” to calculate the set operations and validate whether the operations are equal or not. It has an easy menu driven interface to take user input and validate the Set theorems (both basic and compound operations).

Platforms/Languages: Java, OOPS, Data structures (Hash set), BlueJ Java Editor.

Key Characteristics:

  • Supports core set operations (Union, Intersection, Not)
  • Supports compound set operations (For example: NOT(AUB) = NOT(A) n NOT(B))
  • Uses Java Set library (Hash sets)
  • Menu driven, extensible architecture, modular approach, reusable functions
